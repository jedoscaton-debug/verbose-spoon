# **App Name**: FlowBoard

## Core Features:

- Project & Task Management: Users can create, view, edit, and delete projects. Within each project, they can create and manage individual tasks, assigning names, descriptions, and deadlines.
- Visual Task Boards: An intuitive Kanban-style board interface allowing users to organize tasks across configurable status columns (e.g., 'To Do', 'In Progress', 'Done') using drag-and-drop.
- Dynamic Task Updates: Satisfying micro-animations to visually confirm task status changes, drag-and-drop actions, and task completion, providing immediate feedback.
- Project Progress Dashboard: A centralized dashboard providing a visual overview of key project metrics, including completion rates, upcoming deadlines, and task distribution.
- AI Task Refiner Tool: A generative AI tool that assists users in breaking down large, complex tasks into a set of actionable, smaller sub-tasks based on the initial task description and project context.

## Style Guidelines:

- Primary color: A deep, professional teal (#1FA3A3) to highlight active elements, interactive components, and primary actions.
- Background color: A very light, cool-tinted gray (#ECF6F6) providing a clean, spacious canvas for project boards and content, aligning with the requested 'light gray' scheme.
- Accent color: A vibrant, clear green-cyan (#0DF27A) for highlighting urgent tasks, positive feedback, and calls to action, chosen for its strong contrast and analogous relationship to the primary teal.
- Headline and body font: 'Inter', a grotesque-style sans-serif, providing excellent readability and a modern, objective feel suitable for detailed task management.
- Clean, contemporary vector icons. Prioritize clarity and simplicity for task actions, project status indicators, and navigation elements to ensure intuitive visual communication.
- Utilize a responsive, multi-column Kanban board layout for task organization. Emphasize whitespace and clear demarcation between project sections, statuses, and tasks for enhanced readability and easy drag-and-drop interactions.
- Implement subtle, purposeful motion effects throughout the app. Examples include smooth transitions for drag-and-drop operations, satisfying task completion animations (e.g., fades, soft scales), and gentle growth effects for dashboard progress indicators to enhance user engagement and feedback.
